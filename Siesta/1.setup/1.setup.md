# 1. 환경 설정

SIESTA는 각 계산 노드에 Singularity 컨테이너(`/singularity/siesta_5.4.2.sif`)로 설치되어 있다.  
이 단계에서는 후처리(시각화·분석)에 필요한 Python 패키지만 설치한다.

---

## 자동 설치 (권장)

```bash
bash 1.setup/setup.sh
conda activate summer
```

스크립트가 `summer` 환경을 만들고 필요한 패키지를 모두 설치한 뒤 설치 결과를 확인한다.

---

## 수동 설치

```bash
conda create -n summer -c conda-forge python=3.11 sisl ase matplotlib numpy scipy netcdf4 -y
conda activate summer
```

설치 확인:

```bash
python -c "import sisl; print(sisl.__version__)"
python -c "import ase; print(ase.__version__)"
```

---

## SIESTA 버전 확인 (Singularity)

```bash
singularity exec /singularity/siesta_5.4.2.sif siesta --version
```
